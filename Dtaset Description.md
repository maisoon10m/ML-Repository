# Ecommerce Customers Dataset
## Overview  
The Ecommerce Customers dataset contains behavioral and demographic records for 500 customers of a US-based online retail company that operates both a mobile application and a website. Each record captures how a customer interacts with the platform — including time spent on sessions, app usage, website usage, and membership duration — along with the total amount they spend annually. The dataset is clean, fully populated, and well-suited for supervised regression tasks.
The primary objective when working with this dataset is to build a predictive model that estimates a customer's Yearly Amount Spent based on their engagement patterns. This makes it a practical use case for Linear Regression, feature engineering, and model evaluation techniques  

## Feature Descriptions  
**1.Email:** Used as an identifier only — not used in modeling.  
**2.Address:** Customer mailing address. Not used in modeling due to high cardinality.  
**3.Avatar:**	Color of the customer's avatar on the platform. Not used in modeling.  
**4.Avg.:** Average duration of an in-store style advisory session (in minutes).  
**5.Time on App:** Average time the customer spends on the mobile application per session (in minutes).  
**6.Time on Website:** Average time the customer spends on the website per session (in minutes).  
**7.Length of Membership:**	Number of years the customer has been a registered member of the platform.  
**8.Yearly Amount Spent:**	(Target)	Total amount spent by the customer on the platform in a year (USD). This is the variable we aim to predict.
