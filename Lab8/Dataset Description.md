# KNN Project Dataset

## Overview
The dataset consists of numerical features with anonymized column names describing different attributes of observations.  
The objective is to predict the **TARGET CLASS**, which represents a binary outcome (0 or 1).  
This is a clean dataset with no missing values, making it suitable for a **binary classification task** using algorithms like K-Nearest Neighbors (KNN).

---

## Target Variable
- **TARGET CLASS**:  
  - `0` → Class 0  
  - `1` → Class 1  

---

## Task
The goal is to build a model that can:
> Predict the correct class (0 or 1) based on the given features.
